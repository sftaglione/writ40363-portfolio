Project Description:
IDK You Pick is an interactive, browser-based decision-making tool that lets users add custom options, spin a dynamic wheel, and get a random choice. The app stores options and past picks using localStorage, visualizes category stats through a bar chart, and includes a fully animated spinning wheel built with HTML, CSS, and JavaScript.
